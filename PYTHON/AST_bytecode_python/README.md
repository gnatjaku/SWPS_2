
# Projekt demonstracyjny: Python, bytecode, AST i meta-programowanie



Struktura:

- `README.md` – ten opis
- `01_dis_bytecode.py` – prosty pokaz bytecode (moduł `dis`)
- `02_ast_basic.py` – podstawy AST: parsowanie i wypisywanie drzewa
- `03_ast_walk.py` – przechodzenie drzewa AST (`ast.walk`)
- `04_ast_transform.py` – modyfikacja AST i wykonanie zmienionego kodu
- `05_interpreter_notes_pl.md` – notatki o interpreterze Pythona, VM i kompilacji
- `extras/` – dodatkowe przykłady (lambda + dis, prosty benchmark, exec/compile)
